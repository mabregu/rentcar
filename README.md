# CarRental
> Application to rent vehicles 

## carental-api
### routes

| Endpoint | HTTP | Description |
| --- | --- | --- |
| `/rentals` | GET | Retrieve all the rentals |
| `/rentals/:id/customer` | GET | Retrieve a specific rent with `id` customer |
| `/manufacturers` | POST | Create a new manufacturer |
